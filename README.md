# 🎮 Helix Clicker

A clicker-style game where you open packs, collect cards, and evolve your DNA in a survival-inspired system.

👉 **Play now:**
https://seniorkal.github.io/HelixClicker/

---

## 🧠 About the Project

**Helix Clicker** is a progression-based idle/clicker game inspired by loot box systems and survival mechanics.

You collect cards of different rarities, upgrade them, and generate DNA — the core resource used to unlock new content and evolve your collection.

---

## ✨ Features

* 🎴 Card system with multiple rarities
  *(Common, Rare, Epic, Legendary)*
* 🎁 Pack opening system (loot box style)
* 🧬 DNA generation
  *(active clicking + passive income)*
* 🗺️ Interactive mission system *(in development)*
* 🛒 Shop system

---

## 🎮 Gameplay Loop

1. Open packs
2. Collect and upgrade cards
3. Generate DNA
4. Unlock missions and new rewards
5. Repeat and optimize your progression
